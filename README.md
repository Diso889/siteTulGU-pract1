# siteTulGU-pract
## Архив лежит в папке sitepract
## О проекте
Мной был создан сайт строительной компании, при разработке использовался **1C-Битрикс**.

## Функционал
- Реализован **личный кабинет**
- Есть **калькулятор** для расчёта стоимости проекта  
- Реализован раздел **аренды техники**
- Заявка направляется **менеджеру** (создал отдельного пользователя с нужными правами и функционалом личного кабинета)
- Менеджер может **просматривать заявки** и **давать ответ клиенту**
- Профили пользователей есть в **архиве**

## Роли
- **Клиент*: создаёт заявки и получает ответы
- **Менеджер*: просматривает заявки и отвечает клиенту

## Сайт запускал через XAMPP
